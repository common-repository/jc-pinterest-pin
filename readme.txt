== Description ==
Jc pinterest pin is the Social Plugin. It Shows the Pinterest Pin Widget in your WordPress Site When Visitors Press the Button. By this You can Show Your Favorite pin Easily . Its Automatically Display When page Load. It will work based on shortcode [jc-pin]


== Installation ==


To install JC Pinterst Pin:

1. Upload the 'jc-pinterest-pin' folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Administer in menu -> Pinterest Pin Settings
